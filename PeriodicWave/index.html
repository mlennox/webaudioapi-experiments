PeriodicWave

The definition of PeriodicWave is

The createPeriodicWave() method of the BaseAudioContext Interface is used to create a PeriodicWave, which is used to define a periodic waveform that can be used to shape the output of an OscillatorNode.

So, what does that mean?

A quick aside - OscillatorNode

An OscillatorNode can generate a range of pitched waveforms which are usually output to generate audio. They can also be used to modulate other WebAudio objects, but we'll not touch on that here.

In its simplest form the OscillatorNode generates a sine-wave at a specified frequency. There are built in options to generate square, sawtooth, and triangle each with its own specific sound.
